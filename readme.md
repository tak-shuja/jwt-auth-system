# JWT Auth System

A backend authentication system built with Node.js and Express implementing JWT-based authentication using access and refresh tokens.

## Status

Work in progress. Core authentication logic is still being implemented.

## Current Implementation

- Basic Express server setup
- User routes and controllers
- Sample user data
- SQLite database initialization

## Planned Features

- User registration and login
- JWT authentication
- Access and refresh tokens
- httpOnly cookie-based authentication
- Protected API routes

## Tech Stack

- Node.js
- Express
- SQLite

## Project Structure

```
src/
  controllers/
  routes/
  db/
  data/
  app.js
server.js
```

## Installation

npm install  
npm start
